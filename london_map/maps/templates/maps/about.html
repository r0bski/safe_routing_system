Safe Route Finder

This website uses police crime data to plot routes through London to help predestrians avoid areas of high crime. Enter the address of the start location and destination in the path finder tool and it will generate the shortest and safest paths between those locations. A balanced path will also be generated which will be safer than the shortest path and shorter than the safest path. It will be up to you which path you choose.

Note that crime can be unpredictable and this website can not guarantee your saftey while using it.

If you ever feel unsafe or in danger please call the emergency services on 999 immediately.

Crime Heatmap

The Crime Heatmap tool shows the locations of all the crimes in the data set. The higher the intensity, the more dense the crime is in that area. Note that the intensity is also proportional to the zoom of the map.

Temporal Analysis

The Temporal Analysis tool shows how the number of crimes in London has changed over time. You can use the buttons to filter for specific crimes.
