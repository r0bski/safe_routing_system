# safe_routing_system
## Description
This is a Django Website written by Robert Bulcock
This website uses police data to find safer routes for pedestrians in London by avoiding streets that are recored as having a higher volumne of crime.

## Usage:
1. For best results use VS Code.
2. Open the safe_routing_system folder
3. Open a new terminal and type: 
    - cd london_map
4. Enter the following command in the terminal: 
    - python manage.py runserver
## Note
Police data not included in git code. You can find the data here https://data.police.uk/
